
# PTT 熱門話題與關鍵字分析系統

## 專案簡介
本專案以 Python 製作，具備下列功能：
- 熱門文章爬取
- 關鍵字統計
- 文字雲生成

## 執行方式
1. 安裝依賴：
   ```powershell
   .venv\Scripts\activate
   pip install -r requirements.txt
   ```
2. 執行主程式：
   ```powershell
   python main.py
   ```

## 依賴套件
- requests
- beautifulsoup4
- pandas
- matplotlib

## 目標
協助使用者快速掌握 PTT 熱門話題與關鍵字。
